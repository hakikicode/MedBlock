To Run The project

Start the web2 server 

```
cd backend
npm install
cd server
node server

```
In new terminal
Start the web3Authentication for Patients

```
cd web3Authentication
npm install
npm run Dev
```

In New terminal
Start The project

```
cd Frontend
npm install
npm run Dev
```
